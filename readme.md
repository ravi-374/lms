## Library Management System

## Installation Step

Clone a Project

- **Checkout `frontend`  branch**
- **Composer Install**
- **Setup .env file**
- **Run migration `php artisan migrate`**
- **Run seeder `php artisan db:seed`**
- **Create virtual host with name `local.lms.com`**
- **npm install**
- **npm run dev**
- **Run `local.lms.com`**
